==============================
The main MetaLattice namespace
==============================

.. currentmodule:: metalattice

The main ``metalattice`` namespace has very few objects in it by design.

The public attributes are:

================== =====================================================
``__version__``    MetaLattice version string
``max_cpus``       The maximum number of CPUs to use for multiprocessing
