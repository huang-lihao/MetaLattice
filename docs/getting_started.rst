Installation
------------

.. panels::
    :card: + install-card
    :column: col-lg-6 col-md-6 col-sm-12 col-xs-12 p-3

    Working with conda?
    ^^^^^^^^^^^^^^^^^^^

    MetaLattice is part of the `Anaconda <https://docs.continuum.io/anaconda/>`__
    distribution and can be installed with Anaconda or Miniconda:

    ++++++++++++++++++++++

    .. code-block:: bash

        conda install -c lihaohuang metalattice

    ---

    Prefer pip?
    ^^^^^^^^^^^

    MetaLattice can be installed via pip from `PyPI <https://pypi.org/project/metalattice/>`__.

    ++++

    .. code-block:: bash

        pip install metalattice

    ---
    :column: col-12 p-3
