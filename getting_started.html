Getting started#

Installation#

Working with conda?

MetaLattice is part of the Anaconda distribution and can be installed with Anaconda or Miniconda:

conda install -c lihaohuang metalattice

Prefer pip?

MetaLattice can be installed via pip from PyPI.

pip install metalattice

In-depth instructions?

Installing a specific version? Installing from source? Check the installation page.
